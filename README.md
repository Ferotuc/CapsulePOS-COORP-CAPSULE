# CapsulePOS – Punto de Venta COORP CAPSULE

CapsulePOS es un sistema de punto de venta diseñado para la empresa COORP CAPSULE, dedicada a la venta de accesorios de computadoras.

El sistema permite gestionar:

- Registro de ventas
- Control de inventario
- Gestión de productos
- Reportes básicos de ventas

Este proyecto simula la documentación profesional de un sistema real utilizando Jira y GitHub.
